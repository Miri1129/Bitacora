Colores en CSS

Puedes poner color de varias maneras en CSS:

  • Nombre de color: color: red;
  • Hexadecimal: color: #ff66aa;
  • RGB: color: rgb(255, 102, 170);
  • RGBA (con transparencia): color: rgba(255, 102, 170, 0.5);
  • HSL: color: hsl(330, 100%, 70%);
  • HSLA (con transparencia): color: hsla(330, 100%, 70%, 0.5);
